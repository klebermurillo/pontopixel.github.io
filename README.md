# Pontopixel – Website Oficial

Este repositório contém o código-fonte do **site da empresa Pontopixel**.

O projeto foi desenvolvido com o objetivo de representar a identidade da empresa na web, apresentando seus serviços, trabalhos realizados e informações institucionais. Além disso, o site também é utilizado como **portfólio pessoal**, demonstrando minhas habilidades em criação e desenvolvimento de sites personalizados.

## Funcionalidades

-  Apresentação institucional da Pontopixel  
-  Portfólio com projetos e trabalhos desenvolvidos  
-  Aba exclusiva para **solicitação de orçamentos**, facilitando o contato com clientes  
-  Layout responsivo, adaptado para diferentes dispositivos  

## Objetivo do Projeto

Este projeto serve tanto como:
- Site oficial da **Pontopixel**
- Portfólio profissional para apresentar conhecimentos em:
  - Desenvolvimento web
  - Design de interfaces
  - Criação de sites sob medida para clientes
